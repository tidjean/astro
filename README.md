# Astro (React + TypeScript)

Ce projet a ete migre d'Angular vers **React + TypeScript** avec **Vite**.

## Demarrer le projet

```bash
npm install
npm run dev
```

Application disponible sur `http://localhost:5173/`.

## Scripts utiles

```bash
npm run dev      # mode developpement
npm run build    # build production
npm run preview  # previsualiser le build
```

## WSL2 + NTFS

Le rafraichissement a chaud est configure en mode polling dans Vite pour fonctionner correctement meme quand le projet est sur une partition NTFS montee dans WSL2.

## Stack

- React 19
- TypeScript
- Vite
- Bootstrap 5
